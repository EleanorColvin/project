How to Get Into Reading

Reading is definetly difficult to get used to, especially if you don't feel like you enjoy it, but it can be a great hobby once you get into it

Here's some resources to help you out:

  • The Brooklyn Tech Library - explore the catalog and use resources like Sora
  • Use Goodreads to read book reviews, write reviews, and get personalized recommendations
  • If you have a Brooklyn Public Library or New York Public Library card, you can use libby to borrow books online
  • Similar to Goodreads, StoryGraph can help to reccomend you books based on personal preference

Reading-Related Clubs at Tech:

  • Readers' Delight
  • Feminist Book Club
